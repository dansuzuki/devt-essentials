# RedHat Stuffs

## Version
```
cat /etc/redhat-release
```
## Limits
```
/etc/security/limits.conf
```

## Yum Proxy
Edit /etc/yum.conf
```
proxy=http://x.y.z.0:pppp
```

## RPM with Proxy
```
rpm --httpproxy x.y.z.0 --httpport pppp
```
